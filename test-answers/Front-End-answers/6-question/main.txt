
Лично по мне лучший вариант использовать отдельную таблицу стилей специально для печати.
Также установить ее атрибут media на print.

Данный вариант не будет работать на всех носителях.
Поэтому как вариант можно сделать


<link rel="stylesheet" href="all.css" media="screen, print" type="text/css">

Это нужно, чтобы объединить стили экрана и печати в одной таблице стилей, а затем использовать медиа запросы в этой таблице стилей.


Если решать строго по заданию
То нужно просто добавить в таблицу стилей:

@media print
    {
        .no-print, .no-print *
        {
            display: none !important;
        }
    }

Также если в целом говорить о странице печати, можно сделать ещё такой вариант:
<style>
    @media print {
        body {
            visibility: hidden;
        }
        .print {
            visibility: visible;
        }
    }
</style>

То есть мы полностью убираем все элементы кроме тех, которым мы дали класс .print
