Используя информацию из интернета я нашёл лучшее руководство на свой взгляд.

https://dev.1c-bitrix.ru/user_help/store/sale/settings/import/import_csv.php

Поскольку файлы в формате CSV не содержат информации о структуре каталога и свойствах товаров внутри него,
то перед загрузкой данных необходимо создать структуру торгового каталога.


Для создания структуры выполните следующие действия.

Создайте и настройте требуемый информационный блок, если его ещё нет.

Создайте в информационном блоке все необходимые свойства (для элементов информационного блока должны быть заданы те же свойства,
что и для товаров, импортируемых из CSV файла).

Настройте информационный блок на использование в качестве торгового каталога.

Загрузите иллюстрации товаров (если они есть) на сайт.

Инструкция более подробная:

1. При импорте файла прайс-листа в открывшемся окне можно задать следующие параметры:
    1. Тип загрузки
    2. Первая строка содержит заголовки таблицы
    3. Раздел для новых товаров
    4. Группа для новых товаров

    Указывается в том случае, если товары должны быть связаны с группой, отличной от группы «Товар».
    Выберите действие над товарами:
    Добавить — все товары из прайса будут добавлены в базу;
    Обновить — будет обновлена информация о товарах, которые уже есть в базе;
    Обновить по полю;

    5. Обновить в разделе

    6. Товары, отсутствующие в файле импорта

    Указать действие над товарами уже находящимися в каталоге интернет-магазина, которых нет в файле импорта:

    Не изменять;
    Выставить статус «В резерве»;
    Статус «Доступен», «Под заказ» и «Новинка» сменить на «Нет на складе»;
    Удалить из магазина.

2. Указание соответствие полей импортируемого файла полям на страницах карточек товаров.
Для всех столбцов импортируемого прайс-листа необходимо сопоставить поля в карточке товаров используемых в интернет-магазине.

3. Импорт прайс-листа

4. Импорт прайс-листа с картинками

5. Обновление информации о товарах по заданному полю

6. Импорт характеристик товаров (Групповых полей)
